# CZARNA LISTA IV — OIT
## Kombinacje bezwzględnie zakazane

### FIZYCZNE — NIGDY przez tę samą linię

| Para | Czas | Alternatywa |
|------|------|-------------|
| **Ceftriakson + CaCl₂ / Ca-glukonolan** | Natychmiast | Wapń → wenflon lub inny lumen |
| **Fenytoina + G5%** | Natychmiast | Tylko NaCl 0,9% ≤10 mg/ml |
| **Fenytoina + cokolwiek w linii** | Natychmiast | Absolutnie dedykowana linia |
| **Pip+tazobaktam + amikacyna** | Natychmiast | Amikacyna → wenflon |
| **Amikacyna + jakikolwiek beta-laktam** | Natychmiast | Amikacyna → dedykowana linia |
| **Katecholaminy + NaHCO₃** | Sekundy | NaHCO₃ → własna linia |
| **Amiodaron (wlew ciągły) + NaCl** | Minuty | Tylko G5 + non-PVC |
| **Kaspofungina + G5% / G10%** | Natychmiast | Tylko NaCl 0,9% |
| **Diazepam + NaCl** | Natychmiast | Nierozcieńczony lub G5 |
| **Monofer + G5%** | Szybko | Tylko NaCl 0,9% |
| **Fibryga / Octaplex + inny lek IV** | Natychmiast | Dedykowana linia — bez wyjątku |

### MATERIAŁOWE — NIGDY z zestawem PVC

| Lek | Strata dawki | Wymagany zestaw |
|-----|-------------|-----------------|
| **Nitrogliceryna** | 50–80% | Non-PVC specjalny |
| **Amiodaron** | Adsorpcja + DEHP | Szkło / non-PVC + folia |
| **Nimodypina** | Adsorpcja + DEHP | Zestaw producenta |
| **Diazepam** | 80–90% | Szkło / non-PVC |

### PROCEDURALNE — NIGDY w ten sposób

| Sytuacja | Ryzyko | Zasada |
|----------|--------|--------|
| **Bupiwakaina IV** | NZK oporne | Tylko zewnątrzoponowa/obwodowa. Intralipid w pobliżu. |
| **Klindamycyna — bolus** | NZK | Pompa. Min 600 mg/60 min. |
| **Fenytoina >50 mg/min** | Asystolia | Pompa z limitem. EKG ciągłe. |
| **Teofilina — szybki bolus** | Arytmia/drgawki | Dawka ładująca ≥30 min. |

### FARMAKOKINETYCZNE — niezależne od linii

| Para | Konsekwencja | Twoje działanie |
|------|-------------|-----------------|
| **Meropenem + VPA** | VPA ↓ 60–100% → napady | Monitoring VPA, powiadom lekarza |
| **Flukonazol + midazolam** | 3–5× wzrost sedacji | Oceń RASS po 2–4h |
| **Amiodaron + haloperydol** | TdP → NZK | EKG przed haloperydolem |
| **Amiodaron + digoksyna** | Toksyczność digoksyny | TDM digoksyny |

---

**ZASADA:** Brak danych ≠ bezpieczne. W wątpliwości → lekarz zlecający.
